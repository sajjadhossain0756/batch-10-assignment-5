What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing JavaScript to interact with and manipulate HTML and XML documents. Through the DOM, you can dynamically change content, structure, and style on the page.

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript methods like `document.getElementById()` for selecting by ID, `document.querySelector()` for selecting the first match of a CSS selector, or `document.getElementsByClassName()` for selecting elements by class. These methods allow you to access and manipulate elements on the webpage.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events for its child elements. It is useful because it reduces the number of event listeners needed, improves performance, and ensures that dynamically added elements can still trigger the event without additional listeners.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute() and removeAttribute() to change or remove attributes. To modify styles, you can access the style property (e.g., element.style.color = 'blue';). Additionally, classes can be managed using classList methods such as add(), remove(), or toggle()
